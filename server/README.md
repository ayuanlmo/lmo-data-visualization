## 介绍

这是与 <a href="https://github.com/ayuanlmo/lmo-data-visualization">lmo-Data-Visualization</a>配套的服务端。 

项目代号：   Ying(颖)

## 开始

*合成需要ffmpeg支持，<a href="https://ffmpeg.org/download.html">点击这里下载</a>

服务端默认启动端口为：[3000] 

```bash
#安装模块
yanr
#启动
yarn start-server
```
